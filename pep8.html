About pep8

I am using pep8 in my project to check the style guide for Python code

The first comment I'he got during my first code review, you have W293 pep8 errors and then i thought what is pep8 and why we use pep8?

PEPs are Python Enhancement Proposals, and they describe and document the way python language evolves. They also provide a reference point for the pythonic way to write code. The PEP8 is the Style Guide for Python Code, and it covers:

  • formatting
  • comments
  • naming conventions
pep8 aim to improve readability and reliability of your code.
